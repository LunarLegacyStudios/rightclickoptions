### ⚠️ False Positive Notice

### Features
* **Force Delete:** Permanently deletes a file and kills any active processes locking it. Bypasses the Recycle Bin.
* **Open with Notepad:** Force-opens any file (including .exe or .dll) directly in Notepad to view headers or edit strings.
* **Copy Full Path:** Instantly copies the absolute file path to your clipboard.
* **Take Ownership:** Grants you full admin permissions over a file or folder. (Note: This is currently a little bugged).
